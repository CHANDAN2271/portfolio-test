]]> So usually an image is outputted through a filepath in the ‘src’ attribute in an ‘img’ tag. Although we can manipulate the filepath in such a way we can actually recreate the image by getting the image’s data into base64 encoding. It helps protect outsiders from knowing your folder structure.

Note: IE7 and lower don’t support Data URIs Doing it this way also helps speed up page speed as there is no need to search for a url.

// A few settings
$image = 'cricci.jpg';

// Read image path, convert to base64 encoding
$imageData = base64_encode(file_get_contents($image));

// Format the image SRC:  data:{mime};base64,{data};
$src = 'data: '.mime_content_type($image).';base64,'.$imageData;

// Echo out a sample image
echo '<img src="', $src, '">';
