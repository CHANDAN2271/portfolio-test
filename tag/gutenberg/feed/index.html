]]> Don’t like the new Gutenberg editor? Since WordPress 5.0, the WP core has been updated with the Gutenberg editor.

The Gutenberg editor can be turned off thankfully, by adding a few lines of code to your theme’s function.php file.

/* Disable Gutenberg */

// disable for posts
add_filter('use_block_editor_for_post', '__return_false', 10);

// disable for post types
add_filter('use_block_editor_for_post_type', '__return_false', 10);
